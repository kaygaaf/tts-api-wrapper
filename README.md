# TTS API Wrapper 🎙️

Free Text-to-Speech API wrapper powered by ElevenLabs technology.

## Quick Start

```bash
pip install flask requests
python app.py
```

## API Usage

```bash
curl -X POST http://localhost:5000/api/tts \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello world"}'
```

## Features

- 🚀 Simple REST API
- 🎭 29+ voices
- ⚡ Fast response times
- 💰 Free tier available
